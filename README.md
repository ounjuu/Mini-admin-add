## :hammer: 주요기능
### ✅ 상품등록 <br/> 
1. 전체 내용을 입력해야 버튼이 활성화됩니다.<br/>
2. 자체 상품 코드의 중복 검사가 가능합니다. (axios 이용하여 로컬스토리지 데이터와 비교)<br/>
3. 이미지 등록이 가능합니다. 미리보기 후 하단의 테이블로 이미지 추가가 가능합니다. <br/>
